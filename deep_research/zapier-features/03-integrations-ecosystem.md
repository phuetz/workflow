## Premium vs Free Apps

### Free Apps
- Included in all plans including Free tier
- Core productivity and communication apps
- Limited functionality on Free plan (two-step Zaps only)

### Premium Apps
- Available on Professional plan and above
- Include enterprise tools and specialized services
- Examples: Salesforce, HubSpot (premium features), Custom integrations

## Integration Types

### 1. Pre-built Integrations
- Ready-to-use connections with popular apps
- Maintained by Zapier or app developers
- Regular updates for API changes

### 2. Webhooks
- Connect any app that supports webhooks
- Receive data from external services
- Send data to custom endpoints

### 3. API Requests
- Make custom HTTP requests via Webhooks by Zapier
- Support for GET, POST, PUT, DELETE methods
- Handle JSON, XML, and form data

### 4. Custom Integrations (Developer Platform)
- Build private integrations for internal tools
- Publish to Zapier's public directory
- Full control over triggers and actions

## Developer Platform

### Building Integrations
- Create custom integrations for any app with an API
- Define triggers, actions, and searches
- Test and validate before publishing

### Integration Management
- Version control for integrations

### Private Integrations
- Build integrations for internal use only
- Not published to public directory
- Available on Team and Enterprise plans

## MCP (Model Context Protocol) Integration

### What Is MCP?
- Standardized protocol for AI tools to connect to apps
- Enables AI assistants to take real actions in connected apps

### Capabilities
- Connect Claude, ChatGPT, Cursor to 8,000+ apps
- 30,000+ actions available via natural language
- Handles authentication, rate limits, and retries

### Compatible AI Clients
- Claude
- ChatGPT/GPT-4
- Cursor IDE
- Other MCP-compatible AI tools

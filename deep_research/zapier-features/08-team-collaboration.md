# Team Collaboration Features

## Overview

Zapier provides comprehensive team collaboration features for organizations, enabling shared workflows, centralized management, and role-based access control.

## Team Plans

### Plan Tiers with Team Features

| Plan | Users | Key Features |
|------|-------|--------------|
| Professional | 1 (individual) | No team features |
| Team | Up to 25 | Shared folders, SSO, shared connections |
| Company/Enterprise | Unlimited | Advanced admin, SCIM, custom permissions |

## Shared Folders

### Folder Organization
- Organize Zaps into shared folders
- Group by team, project, or function
- Collaborative access to workflows
- Clear structure for large automation libraries

### Folder Permissions
- Control who can view folders
- Set edit permissions per folder
- Manage access at granular level
- Inherit or override parent permissions

## Shared App Connections

### What Are Shared Connections?
- Team members share authenticated app connections
- Single authentication for entire team
- Reduces duplicate login management
- Centralized credential control

### Benefits
- No need for each user to authenticate separately
- Consistent access across team
- Easier credential rotation
- Simplified onboarding

### Security Considerations
- Connection access controlled by admins
- Audit logs for connection usage
- Revoke access without affecting Zaps
- Enterprise-grade security

## User Roles and Permissions

### Standard Roles

| Role | Capabilities |
|------|-------------|
| Owner | Full account control, billing, all permissions |
| Super Admin | Manage users, settings, all Zaps |
| Admin | Manage team members, shared resources |
| Member | Create and edit Zaps in permitted folders |
| Restricted Member | Limited access as defined by admin |

### Permission Levels
- View only
- Edit
- Create
- Delete
- Manage connections
- Manage members

## Best Practices

### 3. Implement Role-Based Access

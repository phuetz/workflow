# Version History and Unique Features

## Version History

### Zap Versioning
- View previous versions of Zaps
- Track changes over time
- Understand workflow evolution
- Rollback capability

### What's Tracked
- Step additions and removals
- Configuration changes
- Trigger modifications
- Action updates

### Rollback Limitations
- Cannot replay if Zap significantly changed
- Changes include: deleted steps, added steps, moved steps
- Must maintain structural compatibility
- 60-day history retention

### Agent Versioning (New in 2025)
- Copilot updates create checkpoints
- View exactly what was added, removed, or rewritten
- One-click rollback
- Restore specific pieces while keeping others

## Unique Features

### 1. Zapier Canvas

#### What Is Canvas?
- AI-powered workflow diagramming tool
- Map out entire workflows visually
- Identify bottlenecks
- Team alignment on processes

#### Capabilities
- AI-generated process maps
- Logical component organization
- Data flow visualization
- Image support (drag-and-drop, paste)

#### Use Cases
- Process documentation
- Workflow planning
- Team communication
- Bottleneck identification

### 2. Zapier Copilot

#### What Is Copilot?
- AI-powered building assistant
- Works across entire Zapier platform
- Natural language automation creation
- Launched September 2025

#### Capabilities
- Create workflows from descriptions
- Build canvases, tables, interfaces
- Create chatbots and agents
- Suggest solutions and follow-up questions
- Works with 8,000+ integrations

#### How It Works
1. Describe what you want to automate
2. Copilot suggests solutions
3. Asks clarifying questions
4. Builds automation using Zapier tools

### 3. Zapier Agents

#### What Are Agents?
- Specialized AI teammates
- Work independently across apps
- Access live company data
- Autonomous task execution

#### Key Features
- Natural language instructions
- Prompt assistant auto-enhancement
- Live and static data source access
- Granular team sharing permissions
- Browser extension access

#### Capabilities
- Work across 8,000+ app integrations
- Use existing Zaps and Tables
- Interface with Interfaces
- 24/7 autonomous operation

### 4. Central (AI Orchestration)

#### What Is Central?
- Unified AI orchestration platform
- Combines Zaps, Tables, Interfaces
- Agent coordination and governance
- Centralized logging and approvals

#### Benefits
- Single platform for all automation
- Familiar governance model
- Integrated app connections
- Comprehensive audit trail

### 5. Zapier MCP (Model Context Protocol)

#### What Is MCP?
- Standard for AI-app connectivity
- Connect AI tools to 8,000 apps
- 30,000+ available actions
- Natural language execution

#### Compatible AI Tools
- Claude
- ChatGPT
- Cursor IDE
- Gemini
- Other MCP-compatible clients

#### Features
- OAuth-based authentication
- Enterprise-grade rate limiting
- Activity audit logs
- Handles retries automatically

### 6. Sub-Zaps (Reusable Workflows)

#### What Are Sub-Zaps?
- Modular, reusable workflow components
- Call from multiple parent Zaps
- Return results to calling Zap
- Single source of truth for logic

#### Components
1. **Start trigger**: Define input data fields
2. **Actions**: Process steps (unlimited)
3. **Return action**: Send data back to parent

#### Benefits
- Update once, affect all calling Zaps
- Reduce duplication
- Easier maintenance
- Consistent logic across workflows

### 7. Delay and Scheduling

#### Delay by Zapier
- **Delay For**: Pause for X amount of time
- **Delay Until**: Pause until specific date/time
- **Delay After Queue**: Add to queue for sequential processing

#### Schedule by Zapier
- Trigger on recurring schedule
- Hourly, daily, weekly, monthly options
- Custom intervals supported
- Start workflows without external trigger

#### Limitations
- Maximum 30-day delay
- Zap must stay on during delay
- Changes during delay may cause issues

### 8. Storage by Zapier

#### What Is Storage?
- Simple key-value data storage
- Persist data between Zap runs
- No external database needed

#### Operations
- Store value
- Get value
- Increment value
- List values

#### Use Cases
- Track counters
- Store state between runs
- Simple data persistence
- Cross-Zap data sharing

### 9. Lookup Tables

#### What Are Lookup Tables?
- Key-value translation tables
- Map input to output values
- No coding required
- Inline in Formatter step

#### Use Cases
- Translate codes to names
- Map IDs to friendly values
- Region/country translations
- Status code mapping

### 10. Zapier Interfaces

#### Overview
- No-code app builder
- Forms and portals
- Internal tools and apps
- Public-facing pages

#### Components
- Forms (data collection)
- Tables (data display)
- Text blocks
- Images
- Buttons (Zap triggers)

#### Integration
- Direct Tables connection
- Zap triggers from forms
- Authentication options
- Custom branding

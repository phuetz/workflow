2. `Ctrl+Alt+F` - Focus search

1. `Ctrl+A, A` - Select all, then all again (including hidden)

3. `Ctrl+Alt+A` - Select by type
4. `Ctrl+Click+Drag` - Lasso selection

### Efficiency Hacks
1. `Ctrl+R` - Repeat last action
2. `Ctrl+Shift+R` - Repeat with changes
3. `Ctrl+K` - Command palette

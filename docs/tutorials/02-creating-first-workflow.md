# Tutorial 2: Creating Your First Workflow

**Duration:** 10 minutes
**Level:** Beginner
**Prerequisites:** Tutorial 1 (Getting Started)

## Script Summary

### [00:00-02:00] Introduction & Planning
- Explain the workflow goal: Send email when form is submitted
- Show the workflow diagram
- Discuss triggers, actions, and data flow

### [02:00-04:30] Setting Up Webhook Trigger
- Add Webhook trigger node
- Configure webhook path and method
- Test webhook with sample data
- Show webhook URL for integration

### [04:30-07:00] Processing Form Data
- Add Data Transformer node
- Map form fields to email fields
- Use expressions to format data
- Preview transformed data

### [07:00-09:30] Sending Email
- Add Email node (SMTP or SendGrid)
- Configure recipient, subject, body
- Use variables from previous nodes
- Test email sending

### [09:30-10:00] Testing & Deployment
- Test entire workflow with real data
- Check execution history
- Activate workflow
- Next steps

**Key Concepts:**
- Webhooks for external triggers
- Data transformation
- Variable substitution
- Testing workflows
